Расшифровка матриц:
PMU_data_matrix_ef_mutual_angle - варьировался угол между ЭДС систем;
PMU_data_matrix_load - варьировалась величина нагрузки;
PMU_data_matrix_positive_seq - варьировалась величины сопротивления по прямой последовательности (модуль z1, фаза - фиксирована)
PMU_data_matrix_zero_seq - варьировалась величины сопротивления по нулевой последовательности (модуль z0, фаза - фиксирована)
PMU_data_matrix_sc_phase - варьировалася фаза КЗ (0 градусов переход напряжения через 0)
PMU_data_matrix_sc_position - варьировалась точка КЗ
PMU_data_matrix_trans_resistance - варьировалась величина переходного сопротивления в месте КЗ

Пределы варьирования:
улол между ЭДС сисем = [-20, -10, ... , 90] град.;
величина нагрузке в узле системы S2 относительно номинальной нагрузке = [0, 20, ... , 100] %;
величина соопротивления по прямой последовательности системы S1 относительно номинального значения = [10, 20, ..., 200] %; 
величина соопротивления по нулевой последовательности системы S1 относительно номинального значения = [10, 20, ..., 400] %; 
величина фазы КЗ = [0, 10, ..., 360] град.;
точка КЗ = [8, 9, ..., 92] %
переходное сопротивление в месте КЗ = [0.001, 0.101, ... 1), [1, 2, ..., 10), [10, 20, ..., 50] Ом;

Примечание 1: Факторы варировались случайным образом.
Примечание 2: Известно только место замыкания: см. SC_position_1_1000.csv
